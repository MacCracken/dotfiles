# Neon Jungle Hyprland Theme

This configuration is inspired by the **Neon Jungle** aesthetic: a mix of cyberpunk technology, rain-slicked streets, and natural overgrowth.

## ðŸ›  Prerequisites
To get the full look shown in the mockup, ensure you have these installed:
* **Hyprland**: The tiling compositor.
* **Kitty**: Recommended terminal for transparency support.
* **Waybar**: For the top status bar.
* **Hyprpaper**: For setting the wallpaper.
* **Wlogout**: For the themed power menu.

## ðŸš€ Setup Instructions
1. **Wallpaper**: Use a high-resolution version of the "Neon Jungle" mockup. Save it as `wallpaper.png` in your config folder.
2. **Apply Config**: Copy `hyprland.conf` to `~/.config/hypr/`.
3. **Transparency**: The config uses `opacity 0.85`. Ensure your terminal (like Kitty) is configured to allow background opacity.
4. **Color Palette**:
   - **Primary Cyan**: `#00FFFF`
   - **Accent Magenta**: `#FF00FF`
   - **Deep Background**: `#0a0a0f`

Adjust `gaps_out` in `hyprland.conf` to show more or less of the background "overgrowth" wallpaper.
